myfreebook
==========

### 使い方めも

#### 新しいブログページを作る

```sh
hexo new "新しい投稿"
```

#### サーバー起動

```sh
hexo server
```

#### Generate static files

```sh
hexo generate
```

```デプロイ
hexo deploy
```

 * [その他コマンド](http://hexo.io/docs/commands.html)
